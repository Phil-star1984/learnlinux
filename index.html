SSH und Netzwerkzugriff lernen

Eine Übersicht der wichtigsten Befehle und Schritte

1. Verbindung per SSH herstellen

Dieser Befehl ermöglicht es dir, eine Verbindung zu deinem Raspberry Pi herzustellen:

ssh benutzername@hostname_or_ip

Ersetze benutzername durch deinen Benutzernamen und hostname_or_ip durch die IP-Adresse oder den Hostnamen des Raspberry Pi.

2. SSH-Tunnel einrichten

Mit diesem Befehl kannst du einen SSH-Tunnel erstellen, um Zugriff auf die Weboberfläche des Routers zu erhalten:

ssh -L 8080:192.168.1.1:80 benutzername@hostname_or_ip

Dies leitet deinen lokalen Port 8080 auf die IP-Adresse 192.168.1.1 (Router) und den Port 80 weiter.

3. Geräte im Netzwerk scannen

Mit nmap kannst du herausfinden, welche Geräte im Netzwerk aktiv sind:

nmap -sn 192.168.1.0/24

Dieser Befehl führt einen Ping-Scan durch und zeigt alle aktiven Geräte im Netzwerk an.

4. Lokale Ports prüfen

Überprüfe, ob ein bestimmter Port auf deinem Raspberry Pi aktiv ist:

sudo netstat -tuln | grep portnummer

Dieser Befehl zeigt alle laufenden Dienste und den Status des Ports an.

5. Öffentliche IP-Adresse anzeigen

Mit diesem Befehl kannst du die öffentliche IPv4-Adresse deines Geräts anzeigen:

curl ifconfig.me

Für explizite IPv4-Abfrage:

curl -4 ifconfig.me

6. Daten von APIs abrufen

JSON-Daten von einer API abrufen und mit jq formatieren:

curl "https://jsonplaceholder.typicode.com/todos/1" | jq

API-Daten herunterladen und in einer Datei speichern:

curl -X GET "https://api.example.com/data" -o wetter.json

Den Inhalt der Datei anzeigen:

cat wetter.json

API-Daten mit Parametern abrufen und speichern:

curl "https://api.example.com/data?param1=wert1¶m2=wert2" -o ergebnisse.json

API-Daten mit Autorisierung abrufen:

curl -H "Authorization: Bearer YOUR_API_KEY" "https://api.example.com/data" -o ergebnisse.json

Alternative Syntax zum Speichern:

curl "https://api.example.com/data" > ergebnisse.json

7. JSON-Dateien analysieren mit jq

Alle Schlüssel in einer JSON-Datei anzeigen:

jq 'keys' example.json

Die Schlüssel des ersten Elements in einem Array anzeigen:

jq '.[0] | keys' example.json

Alle Pfade in einer JSON-Datei anzeigen:

jq 'paths | map(tostring) | join(".")' example.json

Alle Objekte in einer JSON-Datei iterieren:

jq '.[]' example.json

JSON-Daten in Schlüssel-Wert-Paare umwandeln:

jq 'to_entries[]' example.json

Schlüssel-Wert-Paare mit String-Ausgabe:

jq -r 'to_entries[] | "\(.key): \(.value)"' example.json

Pfade und Werte anzeigen:

jq 'paths | join(".") as $p | getpath($p) | "\($p): \(.)"' example.json
